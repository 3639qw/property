# 부동산등기 챗봇 

법률 정보 접근의 장벽을 낮추기 위해 개발한 AI 기반 부동산등기 질의응답 챗봇 프로젝트입니다.
특히 일반인이 가장 어려워하는 상속등기를 중심으로, 실제 등기 실무에서 사용되는 법령·예규·선례를 기반으로 신뢰도 높은 답변을 제공하는 것을 목표로 합니다.

---

## 📌 프로젝트 개요

* **프로젝트명**: 부동산등기 챗봇
* **개발 목적**: 일반인이 이해하기 어려운 부동산등기·상속등기 법률 정보를 AI로 쉽게 제공
* **핵심 기술**: RAG(Retrieval-Augmented Generation) 기반 법률 문서 검색
* **대상 사용자**: 상속등기, 소유권이전등기 등을 직접 진행해야 하는 일반인

---

## 💡 프로젝트 기획 배경

본 프로젝트는 개인적인 경험에서 출발했습니다.

실제로 상속등기를 직접 진행해야 하는 상황을 겪으며, 다음과 같은 문제를 체감했습니다.

* 관련 법령과 예규가 방대하고 흩어져 있음
* 어떤 서류가 필요한지 명확하게 정리된 자료가 부족함
* 등기관의 판단 기준이 되는 예규·선례를 일반인이 찾기 매우 어려움

이 과정에서 상당한 시간과 시행착오를 겪었고, “일반인이 바로 질문할 수 있는 법률 챗봇이 있다면 얼마나 좋을까?”라는 문제의식이 본 프로젝트의 출발점이 되었습니다.

## 🏛️ 시스템 아키텍처

본 프로젝트는 RAG구조를 기반으로 설계되었습니다.

* 등기 관련 법령·예규·선례 PDF 수집
* 문서 정제 및 청크 단위 분할
* 임베딩 생성 후 벡터 데이터베이스 저장
* 사용자 질문 시 관련 문서를 검색하여 LLM 응답 생성

이를 통해 AI의 환각(Hallucination)을 최소화하고, **법적 근거 기반 답변**을 제공하도록 설계했습니다.

---

## 📚 학습 데이터 (Training Data)

챗봇은 실제 등기 실무에서 활용되는 **공식 법률 문서(PDF)**를 기반으로 학습되었습니다.

### 주요 학습 문서

* 부동산등기법
* 부동산등기규칙
* 상속등기 관련 업무처리 지침
* 대법원 등기예규 및 등기선례
* 가족관계등록법, 호적법
* 민법 및 의용민법

> 모든 문서는 PDF 형태로 수집 후 정제·청크화하여 임베딩에 활용되었습니다.

## 🔑 핵심 기능

### 1️⃣ RAG 기반 법률 질의응답

* 등기관의 판단 기준이 되는 법령·예규·선례를 근거로 답변 생성
* 단순 요약이 아닌 **실무 중심의 설명 제공**

### 2️⃣ 실제 테스트 질문 사례

다음은 실제 챗봇 테스트에 사용된 주요 질문 예시입니다.

* 등기할 수 있는 권리의 종류
* 소유권이전등기 신청 시 첨부 서류
* 상속등기 신청 시 첨부 서류
* 등기명의인표시변경등기의 개념과 적용 사례
* 지상권의 존속기간 및 법적 근거
